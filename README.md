# SYSMATE AR Product Placement

Mobile web AR prototype for placing SYSMATE products in real scale.

## Files

- `index.html`: app entry point
- `css/style.css`: app styles
- `js/app.js`: AR and 3D preview logic
- `manifest.json`: product metadata
- `models/`: GLB product models

For best results, serve this folder over HTTPS. WebXR AR features require a compatible Android Chrome device with ARCore support.
